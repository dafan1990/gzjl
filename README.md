# 极简工作记录系统

一个采用黑色极简设计风格的工作记录管理系统，专注于提供简洁、高效的工作记录体验。

## ✨ 特性

### 🎨 设计风格
- **黑色极简**：采用纯黑背景配合白色文字，创造高对比度视觉效果
- **现代UI**：使用卡片式布局、渐变色彩和精致的交互动画
- **响应式设计**：完美适配桌面端和移动端设备

### ⚡ 交互效果
- **打字动画**：标题区域的动态打字效果
- **鼠标跟随**：优雅的粒子跟随效果（桌面端）
- **滚动渐入**：内容随滚动逐渐显现的动画
- **悬停反馈**：卡片和按钮的精致悬停效果

### 📋 核心功能
- **工作记录管理**：添加、查看、删除工作记录
- **智能筛选**：按日期、关键词筛选记录
- **数据统计**：可视化图表展示工作数据
- **数据导入导出**：JSON格式数据备份和恢复
- **本地存储**：使用localStorage保存数据，隐私安全

### 📊 数据可视化
- **工作内容分布图**：饼图展示不同工作内容的分布
- **每日记录数量**：柱状图展示每日记录数量统计

## 🚀 快速开始

### 直接使用
1. 下载项目文件
2. 用浏览器打开 `index.html` 文件
3. 开始记录你的工作！

### 本地服务器（推荐）
```bash
# 使用Python启动本地服务器
python -m http.server 8000

# 或使用Node.js
npx serve .

# 然后访问 http://localhost:8000
```

## 🛠️ 技术栈

- **前端框架**：原生 HTML5 + CSS3 + JavaScript
- **CSS 框架**：Tailwind CSS v3
- **图标库**：Font Awesome 4.7.0
- **图表库**：Chart.js 4.4.8
- **数据存储**：localStorage API

## 📱 使用指南

### 1. 添加工作记录
1. 点击"添加记录"导航按钮
2. 填写工作日期、标题、描述等信息
3. 可选：添加工作时长和标签
4. 点击"保存记录"按钮

### 2. 查看和筛选记录
1. 在"工作台"页面查看所有记录
2. 使用筛选功能按日期或关键词搜索
3. 点击删除按钮移除不需要的记录

### 3. 生成数据报告
1. 切换到"数据报告"页面
2. 选择开始和结束日期
3. 点击"生成报告"查看统计图表
4. 查看详细的数据表格

### 4. 数据管理
- **导出数据**：点击顶部的"导出"按钮下载JSON文件
- **导入数据**：点击"导入"按钮选择JSON文件恢复数据

## 🔧 自定义配置

### 数据存储
- 数据保存在浏览器的 localStorage 中
- 存储键：`minimal_worklogs`
- 数据格式：JSON数组

### 样式定制
可以通过修改 Tailwind 配置来自定义颜色和样式：

```javascript
tailwind.config = {
    theme: {
        extend: {
            colors: {
                primary: '#你的主色调',
                // 其他颜色...
            }
        }
    }
}
```
